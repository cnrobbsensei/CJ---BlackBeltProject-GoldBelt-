# CJ---BlackBeltProject-GoldBelt-

This game is inspired from Mario and old flash games, being a short 2D platformer cave escape. The game uses two characters from my animations I make, while making sure the characters' personalities are still understood without needing to know context. In reference to many of my animations, some cave art throughout the game features easter eggs, though it is more a fun addition for people who know while keeping the core game playable for those who don't know. The game is built with the idea of not having an enemy or "boss", instead just a character suffering through natural occurrences from their own mistakes. The game introduces a new mechanic in each level, with the fourth level compiling them all into a strenuous challenge. The game is designed with failure in mind. Levels are short enough to where failing allows you to learn from the unpredictable and unfair nature of the cave and adjust yourself.
